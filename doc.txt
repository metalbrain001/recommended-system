postgres common commands.

psql -U metalbrain -h db -d devdb

- List all databases: \l

- Switch to specific database: \c your_database_name

- Show all tables: \dt

- Show all columns of a specific table: \d table_name

- List roles: \du

- Quit : \q

- Get current User and current database: SELECT current_database(), current_user;


Using Quieries:
- SELECT datname FROM pg_database;


View all tables in the current database using a SQL query:
- SELECT table_name FROM information_schema.tables WHERE table_schema='public';

Select all data from a table:
- SELECT * FROM table_name;
